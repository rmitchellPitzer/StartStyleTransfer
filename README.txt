Hello!

The first thing to do if you don't have python installed is to go to
https://www.cs.hmc.edu/twiki/bin/view/CS5/OwnMachines
and follow their tutorial for getting python up and running on your machine!
You'll also need to follow the instructions to install Anaconda/Jupyter.
After that, extract this zip into a directory, and open jupyter notebook by
(on windows)
pressing the windows key + R key, and typing jupyter notebook. Press enter
(on mac)
opening a terminal window from spotlight and typing jupyter notebook. Press enter

Navigate to the folder you extracted this zip to, and open the file named
startStyleTransfer!
